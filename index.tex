\chapter{Modelo Matemático para Optimizar la Operación de un Sistema
Hidroeléctrico.}\label{modelo-matemuxe1tico-para-optimizar-la-operaciuxf3n-de-un-sistema-hidroeluxe9ctrico.}

El objetivo de este capítulo es presentar una formulación rigurosa del
problema de optimización de la operación de un sistema hidroeléctrico
como un Proceso de Decisión de Markov (MDP), junto con una descripción
detallada del algoritmo de Fitted Q-Iteration (FQI) utilizado para
aproximar la política óptima. Se enfatizará la conexión entre los
elementos matemáticos formales y su implementación numérica, así como la
validación operativa mediante simulación en lazo cerrado.

\section{Formulación como Proceso de Decisión de Markov
(MDP)}\label{formulaciuxf3n-como-proceso-de-decisiuxf3n-de-markov-mdp}

Siguiendo la notación de Puterman (1994), se define el MDP como una
\(4-\)tupla \((T, \mathcal{S}, \mathcal{K}, \mathbb{P}, \mathcal{R})\),
donde cada componente se especifica a continuación:

\subsection{Estructura Temporal}\label{estructura-temporal}

Se considera un horizonte temporal de operación compuesto por \(T = 50\)
años históricos, en donde cada uno de ellos se dividieron en \(q = 24\)
quincenas. Con el fin de reducir la dimensionalidad temporal y poder
capturar la estacionalidad, se agruparon las quincenas en \(M = 6\)
etapas hidrológicas (segmentación temporal del año) mediante la
siguiente función de agregación
\(\mu: \{0,\dots,q-1\} \to \{0,\dots,M-1\}\) definida por
\begin{equation}\protect\phantomsection\label{eq-etapas}{\mu(q) = \begin{cases} 0, & 0 \leq q <10\\ 1, & 10 \leq q <14\\ 2, & 14 \leq q < 16\\ 3, & 16 \leq q < 18\\ 4, & 18 \leq q < 20\\ 5, & 20 \leq q < 23\\ \end{cases}}\end{equation}
así, la variable \(m = \mu(q)\) representará la etapa hidrológica
correspondiente a la quincena \(q\). El índice temporal completo se
denotará por como \(\tau = (t,q)\) con \(t \in \{0,\dots,T-1\}\) y
\(q \in \{0,\dots,q-1\}\).

\subsection{Espacio de estados y operador de
discretización}\label{espacio-de-estados-y-operador-de-discretizaciuxf3n}

El sistema está compuesto por dos presas:
\href{https://maps.app.goo.gl/NHNQdHdfYAoRGxhPA}{La Angostura}
\((i = 1)\) y \href{https://maps.app.goo.gl/acX7hVXswX5uYYUBA}{Malpaso}
\((i =2)\). El volumen de almacenamiento de cada una de ellas en la
quincena \(q\) del año \(t\) se representará por
\(V_{i,t,q} \in \mathbb{R}_{\geq 0}\) \((Mm^3)\).

Dado que los datos históricos se utilizan para construir un conjunto de
transión empírica, se introduce un operador de discretización uniforme
del espacio de estados mediante el parámetro \(\Delta = 600\) \(Mm^3\).
Así, el espacio de estados se define como
\(\mathcal{S} = \mathcal{S}_1 \times \mathcal{S}_2 \times \{0,...,5\} = \{s = (s_1,s_2,m)\}\)
en donde \(s_i\) representa el volumen discretizado de la presa \(i\),
tal estado discreto se obtiene mediante el operador de discretización
\(\mathcal{D}: \mathbb{R} \times \mathbb{N} \longrightarrow \{0,\dots,N-1\}\)
dado por
\begin{equation}\protect\phantomsection\label{eq-discretizacion}{s_i = \mathcal{D}(V_{i,t,q}, N_i) = \min \left(\max\left(\left\lfloor \frac{V_{i,t,q}}{\Delta} + \frac{1}{2}\right\rfloor, 0\right), N_i-1\right),}\end{equation}
donde \(N_1 = 27\) y \(N_2 = 17\) son las capacidades discretas máximas
de cada presa.

\subsection{Espacio de acciones}\label{espacio-de-acciones}

Para cada etapa \(m\), se define una unidad de extracción de agua
\(u_m = \{60, 150, 300, 300, 300, 150\}\), así la decisión de operación
en la etapa \(m\) corresponderá a los niveles de turbinado \(k_1, k_2\).
Cada nivel de turbinado se multiplica por la unidad de extracción
\(u_m\) para obtener el volumen de agua extraído en cada presa. Por lo
tanto, el espacio de acciones se define como
\[\mathcal{K}_m = \mathcal{K}_{1,m} \times \mathcal{K}_{2,m},\] donde
\(\mathcal{K}_{i,m}\) representa el conjunto de niveles de turbinado
disponibles para la presa \(i\) en la etapa \(m\).

Una acción denotada por \(a = (k_1, k_2)\), representará el nivel de
turbinado aplicado simultaneamente en ambas presas. Por ejemplo, suponga
que \(m = 0\) entonces \[\mathcal{K}_1, \mathcal{K}_2 = \{0,1,...,7\},\]
y así \(a = (3,5)\) significaría que Angostura turbina
\(3\times 60 = 180\) \(Mm^3\) de agua y Malpaso turbina
\(5\times 60 = 300\) \(Mm^3\) de agua.

\subsection{Dinámica de transición}\label{dinuxe1mica-de-transiciuxf3n}

La ecuación de transición continua tras la operación del sistema se
define por
\begin{equation}\protect\phantomsection\label{eq-transicion}{V_{i,t,q+1} = V_{i,t,q} + W_{i, t, q+1}-k_iu_m, \quad q<23}\end{equation}
donde \(V_{i,t,q}\) representa el volumen almacenado en la presa \(i\)
antes de tomar la acción de turbinado y \(W_{i, t,q+1}\) representa el
afluente neto (caudal de ingreso) para la presa \(i\) en la quincena
\(q+1\) del año \(t\).

Cuando \(q=23\), la transición se define por
\begin{equation}\protect\phantomsection\label{eq-transicion23}{V_{i,t+1,0} = V_{i,t,23} + W_{i, t+1, 0}-k_iu_m.}\end{equation}

En ambos casos, el volumen discretizado siguiente se obtiene aplicando
el operador de discretización \(\mathcal{D}\) al volumen resultante, es
decir, \[s_i' = \mathcal{D}(V_{i,t,q+1}, N_i),\] mientras que la etapa
evoluciona mediante \(m' = (m+1) \mod M\).

Por lo tanto, el estado discreto siguiente está dado por
\[s' = (s_1', s_2', m').\]

Desde el punto de vista de Puterman (1994), la transición de MDP se
define mediante una función de probabilidad condicional
\begin{equation}\protect\phantomsection\label{eq-transicion}{\mathbb{P}(s'|s,a),}\end{equation}
la cual especifica la probabilidad de transicionar al estado \(s'\) dado
que se tomó la acción \(a\) en el estado \(s\).

\section{Función de recompensa}\label{funciuxf3n-de-recompensa}

\subsection{Modelo hidráulico-energético
linealizado}\label{modelo-hidruxe1ulico-energuxe9tico-linealizado}

La energía generada depende del volumen de agua turbinado y una
aproximación lineal de la altura hidraúlica. Para cada presa \(i\), se
define la función de altura
\begin{equation}\protect\phantomsection\label{eq-altura}{H_i (s_i,s'_i) = 10 (s_i + s_i'+2)}\end{equation}
tal expresión corresponde a un modelo lineal basado en estados
discretizados, el término \(+2\) se introdujo para evitar alturas nulas
en estados con bajo volumen.

Sea \(\eta = 0.9\) eficiencia de conversión hidráulica a eléctrica, y
\(g = 9.81\) \(m/s^2\) aceleración de la gravedad, entonces la energía
generada por la presa \(i\) al tomar la acción \(a\) en el estado \(s\)
y transicionar a \(s'\) se define por
\begin{equation}\protect\phantomsection\label{eq-energia}{E_i(s,a,s') = \frac{\eta g  H_i(s_i,s'_i) (k_iu_m\times 10^6)}{3600\times 10^6},}\end{equation}
así, la energía total generada por el sistema es
\begin{equation}\protect\phantomsection\label{eq-energia-total}{E = E_1+E_2.}\end{equation}

\subsection{Estructura de
penalizaciones}\label{estructura-de-penalizaciones}

La recompensa incorpora penalizaciones para garantizar el cumplimiento
de restricciones operativas. Se definen las siguientes penalizaciones:

\begin{itemize}
\item
  Penalización por derrame: si \(V_{i,t,q+1} > N_i \Delta\) se aplica
  \begin{equation}\protect\phantomsection\label{eq-penalizacion-derrame}{\Pi_{derr} = C_{derr}\cdot\left(\frac{V_{i,t,q+1}-N_i \Delta}{\Delta}\right)}\end{equation}
\item
  Penalización por déficit de agua: si \(V_{i,t,q+1} < 0\) se aplica
  \begin{equation}\protect\phantomsection\label{eq-penalizacion-deficit}{\Pi_{def} = C_{def}}\end{equation}
\item
  Penalización por violación de curva guía: si \(s_i' < \text{CG}_i(m)\)
  se aplica
  \begin{equation}\protect\phantomsection\label{eq-penalizacion-curva-guia}{\Pi_{CG} = C_{CG}\cdot(s_i'- \text{CG}_i(m))}\end{equation}
\item
  Penalización por niveles críticos: si \(s_i' \geq N_i-1\) se aplica
  \begin{equation}\protect\phantomsection\label{eq-penalizacion-niveles-criticos}{\Pi_{crit} = C_{crit}}\end{equation}
\end{itemize}

donde \(C_{derr}, C_{def}, C_{CG}, C_{crit}\) son constantes de
penalización ajustadas empíricamente, y \(\text{CG}_i(m)\) representa la
curva guía para la presa \(i\) en la etapa \(m\).

La función de recompensa total se define por
\begin{equation}\protect\phantomsection\label{eq-recompensa}{\mathcal{R}(s,a,s') = E - \Pi_{derr} - \Pi_{def} - \Pi_{CG} - \Pi_{crit}.}\end{equation}

En el presente trabajo no se modela una distribución analítica
explícita, por lo que en la práctica se construye un dataset empírico
basado en datos históricos. La dinámica se construye a partir de
trayectorias históricas \[(s, a, r, s')\] en donde cada una de ellas
representa una experiencia de transición observada en los datos
históricos, siendo \(r\) el valor obtenido de la ecuación
\{\#eq-recompensa\}, representando la recompensa obtenida al tomar la
acción \(a\) en el estado \(s\) y transicionar a \(s'\). Este enfoque se
alinea con un esquema de \textbf{\emph{Batch Reinforcement Learning.}}
